Este es un texto de muestra en español para la aplicación de desplazamiento de doble idioma.

La aplicación permite a los usuarios leer contenido lado a lado en dos idiomas diferentes.

Este texto demuestra cómo funciona la sincronización de desplazamiento entre los dos marcos.

Cada párrafo debe alinearse verticalmente al desplazarse, proporcionando una experiencia de lectura fluida.

El texto es simple y sin estilo, como se especifica en los requisitos.

Los usuarios pueden navegar entre diferentes elementos de contenido usando la página de inicio.

El visor muestra dos marcos apilados verticalmente, uno para cada idioma.

El comportamiento de desplazamiento está sincronizado para que ambos marcos se muevan juntos.

No se muestran barras de desplazamiento, pero se preserva la funcionalidad.

La aplicación está optimizada para dispositivos móviles e interacciones táctiles.
